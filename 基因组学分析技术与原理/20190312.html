基因组测序——有难有益。人类基因组有大量重复序列，难在拼接问题。

第一代测序

大规模基因组测序的两种策略

  • Clone by Clone/Bac by Bac
  • Whole Genome Shotgun
两种大规模基因组测序策略的比较
项目 策略
全基因组霰弹法 逐步克隆法
遗传背景 不需要 需要（需构建精确的物理图谱）
速度 快 慢
费用 低 高
计算机性能 高（以全基因组为单位进行拼接） 低（以BAC为单位进行拼接）
适用范围 工作框架图 惊喜图
代表测序物种 果蝇、水稻 人、线虫

工作草稿（框架图）与完成图

  • 框架图
    • 90%
    • 4X
  • 完成图
    • 99.99%
    • 8X

第二项是测序的覆盖度，指测序量是基因组大小的几倍。

人类基因组计划研究的主要成果和进展

  • 遗传图谱

    又称为连锁图谱（linkage map），指基因或DNA标志在染色体上的相对位置与遗传距离

  • 物理图谱

    以定位的DNA标记序列如STS作为路标，以DNA实际长度即bp、kb、Mb为图距的基因组图谱。

  • 转录图谱

    利用EST（expressed sequence tags 表达序列标签）作为标记所构建的分子遗传图谱

  • 序列图谱

    通过基因组测序得到的，以A、T、G、C为标记单位的基因组DNA序列

逐步克隆法过程

  • 物理图谱的构建
  • 大片段克隆的筛选
  • 霰弹法测序与“工作框架图”的构建
  • 序列的全组装与“完成图”构建

物理图谱的制作——序列标签位点（STS）作图

STS是随机地从人类基因组上选择出来的长度在200～300bp左右的特异性短序列。唯一的短片段。

物理图谱构建的步骤
  1. 确定各STS序列及其在基因组中的位置
  2. 大插入片段基因组文库的构建（BAC文库）
  3. 以特定STS为标记筛选并定位克隆
  4. 含有STS的克隆在基因组中排序

文库

文库——含有某种生物体全部基因的随机片段的重组DNA克隆群体

宿主——能容纳外源DNA片段的生物体，常用的有大肠杆菌、酵母等

载体：能携带外源DNA进入宿主细胞的工具，常用的载体有质粒载体、噬菌体载体、细菌人工染色体等

作为载体的基本要求

  • 能在宿主细胞中进行独立的复制
  • 具有多克隆位点，可插入外源
  • DNA片段
  • 有合适的筛选标记，如抗药性
  • 大小合适，易于分离纯化
  • 拷贝数多

载体种类：YAC、BAC、phage、COSMID、FOSMID

噬菌体作载体时，会造成污染，杀死试验用的大肠杆菌，如感受态。可以购买抗phage的感受态细胞。

BAC文库的构建
  1. 脉冲场凝胶电泳得200kb左右的大片段DNA；
  2. 纯化后与载体连接；
  3. 电转化，将连接产物导入大肠杆菌感受态细胞；
  4. 在含有氯霉素的固体培养基中培养；
  5. 每一个菌落为带有相同外源DNA片段的单克隆。
